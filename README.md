### PARKING APPLICATION

This is a simple parking application that allows users to manage parking spaces and vehicles. The application provides functionalities to add, remove, and view parking spaces and vehicles.

#### Features
- Add and remove parking spaces
- Add and remove vehicles
- View all parking spaces and vehicles

### ParkingAppUI class
The `ParkingAppUI` class is responsible for the user interface of the parking application. It provides methods to interact with the user, display menus, and handle user input.
This class uses the `ParkingLot` and `Vehicle` classes to manage parking spaces and vehicles.
#### Methods
- `display_menu()`: Displays the main menu of the application.
- `add_parking_space()`: Prompts the user to add a new parking space.
- `remove_parking_space()`: Prompts the user to remove an existing parking space.
- `view_parking_spaces()`: Displays all parking spaces.
- `add_vehicle()`: Prompts the user to add a new vehicle.
- `remove_vehicle()`: Prompts the user to remove an existing vehicle.
- `view_vehicles()`: Displays all vehicles.

- customers → stores all registered customers

- parkingSpots → stores all parking spots in the system

- reservations → stores all reservations
